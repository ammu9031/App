# 🎶Interactive Music Player
##  About the Project
🎤🎤 This project is a responsive interactive music    
player   built with HTML and CSS that allows user   
to play pause and navigate through song easily .     
it provides a clean UI and smooth user experiences     
making it simple to enjoy music directly in the browser.  
## ✨ Features
🎧play songs using audio player  
▶️play and pause songs  
⏭️skip to next track  
⏮️Go back to previous track  
🔊Volume control   
📂Multiple song list  
🎨Styled using css  
🦚Fast and lightweight
